# Проект: Место

### Обзор учебного проекта
Страница с карточками, которые содержат информацию о местах - фотографию и название места. Карточки можно добавлять, лайкать, удалять. Изменения не сохраняются.
Можно просмотреть увеличенную фотографию в лайтбоксе.
Отображается инфо о профиле, её можно редактировать. 

* https://vadzimasipovich.github.io/mesto/
* JS - формы, попапы, карточки
* Верстка - респонсивная и адаптивная, анимации открытия и закрытия попапов
